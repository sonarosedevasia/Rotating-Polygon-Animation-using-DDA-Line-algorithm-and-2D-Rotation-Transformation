Project Description

This project is a Java-based graphical application that displays a rotating polygon using the DDA (Digital Differential Analyzer) Line Drawing Algorithm. The polygon rotates continuously with dynamic color changes, and users can switch between different shapes using keyboard inputs.

Features

* Supports multiple shapes: Triangle, Square, Pentagon, Hexagon
* Smooth rotation animation
* Dynamic color changes using HSB color model
* Custom line drawing using DDA algorithm
* Interactive keyboard controls

Technologies Used

* Java
* Java Swing
* AWT Graphics

Concepts Used

* DDA Line Drawing Algorithm
* 2D Rotation Transformation

Controls

Press the following keys to change shapes:
1 → Triangle
2 → Square
3 → Pentagon
4 → Hexagon

▶️ How to Run

1. Compile the program:
   javac RotatingPolygonDDA.java

2. Run the program:
   java RotatingPolygonDDA
